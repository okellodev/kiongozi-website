# Kioo Ngozi Leather Website

A luxury artisan storefront dedicated to premium Kenyan leather craftsmanship.

## 🌟 The Kiongozi Spirit
In Swahili, "Kiongozi" means Leader. This website reflects that ethos through a sophisticated aesthetic and a focus on craftsmanship, heritage, and the artisan promise.

---

## 🛠 Technical Stack
- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS
- **Design System:** Custom "Cream & Gold" luxury theme
- **Icons:** Official WhatsApp & Social integration
- **Animations:** Framer Motion for premium interactions

---

## ✨ Key Features
- **Artisan Showcase:** High-resolution product displays with sticky details and Zoom effects.
- **Story-Driven UX:** Integrated "Spirit of Kiongozi" about section and artisan promise.
- **Bespoke Interaction:** Direct WhatsApp connection for custom project requests.
- **Mobile Perfection:** Responsive layouts optimized for high-end mobile devices.
- **HTML Descriptions:** Support for rich product storytelling.

---

## 🏁 Getting Started

First, install dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```

Open [http://localhost:3001](http://localhost:3001) (or your configured port) to see the result.

---

## 🏗 Project Structure
- `/src/app/`: App router and artisan layouts.
- `/src/lib/api.ts`: Dedicated API layer for the headless backend.
- `/public/`: Luxury brand assets and media.
